## What Problem Are We Solving?

A common beginner question is: how much math is actually needed for AI, machine learning, and data science? This playlist answers that by focusing on practical concepts that show up repeatedly in analysis, modeling, and ML workflows.

## Three Basic Things

| Foundation | Simple meaning | Why it matters in AI/ML |
|---|---|---|
| Linear algebra | How AI sees the world | Data becomes vectors, matrices, embeddings, and transformations. |
| Statistics / probability | How AI makes decisions | Used for uncertainty, distributions, descriptive analysis, and model evaluation. |
| Calculus | How AI corrects and learns from mistakes | Used in gradients, optimization, and model training. |

## How To Study This Playlist

A practical sequence for a GitHub-based learning repo:

1. Watch one video.
2. Create one short markdown note.
3. Add one small example or table.
4. Add one Python or pandas snippet where relevant.
5. Add one ML relevance section and one revision section.

This keeps the repo useful both for personal revision and for anyone else reading it later.

## Learning Flow

The early videos build a strong base in descriptive statistics first: center, spread, and distribution. That is a smart order because before training models, it is important to understand what the data looks like, how noisy it is, and whether unusual values are distorting decisions.

## Revision Cheat Sheet

- Learn the essentials first, not the full depth of mathematics.
- Prefer intuition first, formulas second, code third.
- Tie every topic to one real data problem.
- Revisit topics iteratively instead of aiming for perfect understanding in one pass.

## Next Topic Recommendation

After the first lesson on mean, median, mode, and percentile, the next natural topic is standard deviation and mean absolute deviation. Once the center of data is clear, the next question is how spread out the data is.
